How JGA-WES dataset was created (GRCh38)

JGA-WES

We performed variant calling of individual-level exome data deposited at JGA by using the reference sequence GRCh37. Analysis pipeline software (Amelieff BioReT Ver.3.3.1.1) that can execute the following group of analysis tools was used. Additionally, we used transanno to convert GRCh37-based variant data to GRCh38-based one.
